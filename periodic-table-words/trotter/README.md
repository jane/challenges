## Code

See [src/Main.hs](https://github.com/jane/challenges/blob/master/periodic-table-words/trotter/src/Main.hs)

## build/run

Install `stack`:

```sh
brew install haskell-stack
stack setup
```

Build and run

```sh
stack build
stack exec periodic-table-words
```
